# [Schema Weaver](https://schemaweaver.vivekmind.com/) Documentation

Welcome to the official documentation for Schema Weaver. This guide covers everything from connecting your first PostgreSQL database to running migrations, exploring your data, and collaborating with your team.

---

## What is Schema Weaver?

Schema Weaver is a PostgreSQL schema management platform. It gives you a visual SQL Editor for writing and organizing your schema, an auto-generated ER diagram that updates as you type, a migration engine for safely pushing changes to your live database, and an AI-powered Data Explorer for querying and analyzing your data — all in one workspace.

---

## The Two Product Surfaces

| Surface | What it's for | Link |
|---|---|---|
| **SQL Editor** | Write schema, analyze it with the compiler, visualize it as an ER diagram, and manage migrations against a live database | [sql.schemaweaver.vivekmind.com](https://sql.schemaweaver.vivekmind.com/) |
| **Data Explorer** | Browse tables, run SQL queries, analyze columns, and use Resona AI to explore data in natural language | [data.schemaweaver.vivekmind.com](https://data.schemaweaver.vivekmind.com/) |

Both surfaces share the same database connections, projects, and team memberships. You connect a database once and it is available everywhere.

---

## What You Can Do

**Schema Development**
- Write PostgreSQL DDL across multiple files and folders in a structured project
- See your schema graded and analyzed across 20 quality layers in real time
- Visualize every table, view, function, and relationship as an interactive ER diagram
- Use Resona AI to generate SQL, explain schema decisions, and run surgical edits across files

**Database Sync**
- Pull the live schema from any PostgreSQL database into your project files
- Diff your project SQL against the live database before pushing
- Apply migrations safely — with advisory locking, drift detection, and automatic rollback support
- Detect when someone changed the database outside of Schema Weaver

**Data Exploration**
- Browse any table with a high-performance, filterable data grid
- Run custom SQL queries with syntax highlighting and autocomplete
- Analyze columns — null rates, distinct counts, value distributions
- Use Resona AI to write queries, generate charts, and build reports from your data

**Team Collaboration**
- Invite teammates to projects with owner or member roles
- Share database connections without sharing credentials
- Track every file edit and migration by author and timestamp

## Quick Start

1. Open the **[SQL Editor](https://sql.schemaweaver.vivekmind.com/)** and create a new project
2. Add a `.sql` file and write your first `CREATE TABLE` statement
3. Watch the **ER Diagram** and **Schema Compiler** update automatically
4. Connect a PostgreSQL database via the **Connect** button in the footer
5. Run `sw pull` in the terminal to import the live schema, or `sw push --apply` to push your changes
